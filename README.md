## Descripción:
En este proyecto se desarrolló un modelo de procesamiento de lenguaje natural (NLP) para clasificar automáticamente reseñas de películas como positivas o negativas. El sistema fue diseñado para apoyar la moderación y categorización de contenido en una plataforma de análisis de películas.
## Objetivo:
Construir un modelo de clasificación de texto con un valor F1 ≥ 0.85 para identificar reseñas negativas de manera automática.
## Actividades realizadas:
### Exploración y preprocesamiento de datos:
* Limpieza de texto (eliminación de caracteres especiales, normalización).
* Análisis del equilibrio de clases.
### Análisis exploratorio de datos (EDA):
* Evaluación de distribución de reseñas positivas y negativas.
### Preparación de datos para modelado:
* Transformación de texto a variables numéricas (por ejemplo, TF-IDF).
### Entrenamiento de modelos de clasificación:
* Regresión logística.
* Modelos de gradient boosting.
* Otros modelos adicionales para comparación.
### Evaluación de modelos:
* Medición del desempeño mediante la métrica F1.
* Comparación de resultados entre modelos en conjunto de prueba.
### Pruebas adicionales:
* Clasificación de nuevas reseñas creadas manualmente.
* Análisis de diferencias en predicciones entre modelos.
### Exploración opcional de técnicas avanzadas:
* Uso de embeddings (BERT) en subconjuntos de datos.
## Resultado:
Se desarrolló un modelo capaz de clasificar reseñas de texto con alta precisión, alcanzando el umbral requerido de F1. El proyecto demuestra la capacidad de trabajar con datos no estructurados y aplicar técnicas de NLP para resolver problemas reales de clasificación.
